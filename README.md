<h1 align="center">Hi there, I'm Kaung Myat Wai Yan 👋</h1>

I'm a medical doctor turned Health Data Scientist. I build data pipelines, machine learning models, and simulations to solve complex clinical and operational challenges in healthcare. Currently working at NHS England (London Region), I'm passionate about reproducible analytics (RAP) and leveraging tools like Python, Databricks, and PySpark to make sense of messy health data.

### 🤖 Machine Learning
- **CKD Risk Prediction Model**  
  *Supervised machine learning pipeline built in Python to predict undiagnosed Chronic Kidney Disease. Designed to handle highly imbalanced healthcare datasets with a focus on model explainability for clinical interpretability.*
- **Mental Health Services ML Package**  
  *A modular Python library that simplifies training and evaluating classification models on MHSDS data. It abstracts away complex data preprocessing steps and standardizes model benchmarking.*


### 📊 Data Analytics and Engineering
- **Primary Care Reporting**  
  *End-to-end data engineering architecture using PySpark on Databricks to aggregate 20+ disparate data sources, and R to transform data and generate Quarto report tracking 60+ operational metrics.*
- **OMOP Federated Analytics**  
  *Architected federated analytics workflows utilizing the OMOP Common Data Model (CDM) to safeguard patient privacy while expanding research access to comprehensive, distributed healthcare datasets.*
- **IBD Bioresource RAP & Survival Analysis**  
  *Developed a Reproducible Analytical Pipeline (RAP) in Python to automate data ingestion, cleaning and exploratory data analysis for the Inflammatory Bowel Disease dataset of NIHR Bioresource. Used this pipeline and advanced statistical analyses including propensity score matching and survival analysis on a 15k+ patient retrospective cohort, proving early advanced therapy reduces long-term hospitalization risk. Published as first author in Inflammatory Bowel Diseases.*

### 📈 Simulation
- **Healthcare Resource Allocation**  
  *A Python-based simulation engine utilizing the Simulated Annealing optimization algorithm. Built to dynamically solve complex resource distribution problems under strict budget constraints.*
- **Acute Stroke Unit DES Model**  
  *Discrete Event Simulation mapping stochastic patient flow networks in an Acute Stroke Unit. Used to computationally evaluate bottlenecks and optimize long-term bed capacity forecasting.*
